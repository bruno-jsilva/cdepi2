# cdepi2

Projeto de análise em R desenvolvido no GitHub Codespaces para a disciplina "Ciência de Dados Aplicada à Epidemiologia II".

## Estrutura do projeto

- `dados/`: dados brutos, arquivos CSV, planilhas etc.
- `scripts/`: scripts em R, incluindo análise principal e funções.
- `resultados/`: gráficos, tabelas e arquivos processados.
- `docs/`: documentação, relatórios e apresentações.
- `cdepi2.Rproj`: arquivo de projeto R.
- `README.md`: descrição do projeto e instruções de uso.

## Como executar

No terminal, dentro da pasta do projeto, rode:

```bash
Rscript scripts/analise.R
